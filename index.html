Use Offline & Install as an App

  1. Local (desktop) app‑like: save this file as index.html and open in Chrome/Edge. It runs fully offline (no server needed) but service workers (for install prompt) require HTTPS/local server.
  2. Android PWA (recommended): host the three files index.html, manifest.webmanifest, sw.js on HTTPS (e.g., Netlify/GitHub Pages). Then open the link on Android → Chrome menu → Add to Home screen. You’ll get a standalone app with offline cache.
  3. APK/Play Store: wrap this site with Capacitor (WebView) or TWA. Click Android Build Guide below for copy‑paste commands and minimal code.

Note: Install button appears after the browser fires the beforeinstallprompt event (served via HTTPS).
